Differences Between HTML and HTML5

HTML HTML5
Does not support native video or audio embedding. Introduced <video> and <audio> tags to support media without plugins.
Limited in providing meaning through tags. Includes semantic tags like <section>, <article>, <aside>.
No support for offline storage capabilities. Supports local storage and session storage using Web Storage API.
Relied on external technologies for complex graphics. Introduced the <canvas> element for rendering graphics and animations via JavaScript.
